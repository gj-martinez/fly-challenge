# Challange Flybondi
Implementacion de API con grapql con base de datos mongodb


## Instalacion
```
git clone https://github.com/gj-martinez/fly-challenge.git
cd fly-challenge
npm install

```

Crear y configurar .env con los siguientes datos 

```
DB_USER=
DB_PASSWD=
DB_HOST=
DB_NAME=
PORT = 
```


## Iniciar

```
npm start
```
